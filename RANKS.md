# SECTION-97 Rank System

## How XP Works
- Every **₦10,000** spent = **1 XP**
- Levels use quadratic thresholds (gaps get bigger as you rank up)

## Design Notes
- Rank badge color matches the rank's theme
- Achievements display as hexagonal badges on the profile page
- Locked achievements have a dark overlay
- Progress bars show how close users are to unlocking each achievement
